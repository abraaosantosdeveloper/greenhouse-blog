GreenHouse Monitoring System

Sistema de Monitoramento IoT

Este sistema de monitoramento utiliza sensores ESP32 para acompanhar em tempo real as condições da sua estufa. Monitore temperatura, umidade, luminosidade e outros parâmetros importantes para o crescimento das plantas.

A barra de progresso mostra o quanto você já explorou do conteúdo, permitindo uma navegação mais intuitiva pelos dados e configurações do sistema.

Vídeo demonstrativo

Controle Automatizado

Com base nos dados coletados, o sistema pode automaticamente ajustar as condições da estufa para otimizar o crescimento das plantas. Isso inclui controle de irrigação, ventilação e iluminação artificial.

Todos os ajustes são registrados e podem ser acompanhados através desta interface, proporcionando total transparência sobre o funcionamento do sistema.

Alertas e Notificações

O sistema envia alertas quando os parâmetros saem dos valores ideais definidos. Você pode configurar notificações por email, SMS ou push para smartphone.

Isso garante que você seja informado imediatamente sobre qualquer situação que possa afetar suas plantas, mesmo quando estiver longe da estufa. Estes recursos estão em fase de estudos e serão adicionados em atualizações futuras. Por hora, há somente um subscriber via console, desenvolvido com a linguagem Python.

Histórico de Dados

Todos os dados coletados são armazenados em um banco de dados seguro, permitindo análises históricas e identificação de padrões ao longo do tempo.

Você pode gerar relatórios personalizados e exportar os dados para análise externa, ajudando na tomada de decisões baseadas em evidências.

Interface Responsiva

A interface do sistema foi desenvolvida para funcionar perfeitamente em qualquer dispositivo - computador, tablet ou smartphone. Assim você pode monitorar sua estufa de qualquer lugar.

O design responsivo garante que todas as funcionalidades estejam disponíveis independentemente do tamanho da tela utilizada.

Sustentabilidade

O sistema foi projetado com foco na sustentabilidade, otimizando o uso de recursos como água e energia. Isso reduz custos operacionais e o impacto ambiental.

Sensores precisos evitam desperdícios e garantem que as plantas recebam exatamente o que precisam, quando precisam.

Escalabilidade

O sistema é modular e pode ser facilmente expandido com novos sensores e funcionalidades conforme suas necessidades crescem.
